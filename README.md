# PUBG 스팟 메모 & 랭크 분석

배틀그라운드 랭크 게임용 스팟 메모 및 분석 웹 앱.

## 기능
- 공식 PUBG 고화질 맵 11종 내장
- 등급별(S~F) 스팟 마킹 + 호버 팝업
- 고정 차량 스폰 마킹 (드래그 이동)
- 비행선 경로 그리기
- 드래그 서클(자기장) 배치
- 3가지 전략 추천 (등급/안전/서클 우선)
- Firebase 연동 시 나무위키처럼 실시간 공유 저장

## GitHub Pages 배포
1. 이 저장소를 Fork 또는 Clone
2. Settings → Pages → Source: `main` / `/ (root)` → Save
3. 약 1분 후 `https://유저명.github.io/pubg-spot-memo/` 에서 접속 가능

## Firebase 공유 저장 설정 (선택)
`index.html` 상단의 `FIREBASE_CONFIG` 주석 참고.
